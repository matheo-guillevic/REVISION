Sujet blanc guide

Composition type 2 h

Duree : 2 h Calculatrice : selon consigne du cours Objectif : couvrir les 4 chapitres
  1. Denombrement : trois modelisations de tirage, puis probabilite uniforme.
  2. Conditionnement : arbre, probabilites totales, Bayes.
  3. Variable discrete : loi, fonction de repartition, esperance, variance, reconnaissance d'une loi usuelle.
  4. Variable continue : densite, fonction de repartition, esperance, loi exponentielle ou normale.
Methode de correction

Chaque reponse doit nommer les objets, verifier les hypotheses, appliquer une formule citee, puis conclure par une phrase. Les calculs seuls ne suffisent pas.
