Exercice 1

Exercice 1 - Synthèse RST pour un système du 1er ordre

1. Modélisation et cahier des charges

Raisonnement : La fonction de transfert du processus est \(H(p) = \frac{1}{1+p}\). On identifie les polynômes du système : \(A(p) = p+1\) (degré \(n_A=1\)) et \(B(p) = 1\) (degré \(n_B=0\)).

Le cahier des charges impose :

  • Rejet asymptotique des perturbations constantes en sortie : La fonction de sensibilité \(S_{yp}(p) = \frac{A(p)S(p)}{A(p)S(p)+B(p)R(p)}\) doit s'annuler pour \(p=0\). Puisque \(A(0) = 1 \neq 0\), il faut impérativement imposer \(S(0)=0\). On fixe donc une partie de \(S(p)\) : \(S(p) = p \cdot S'(p)\).
  • Pôle de poursuite : \(p_c = -4 \implies P_c(p) = p+4\).
  • Pôles de filtrage : \(p_f = -2 \implies P_f(p) = (p+2)^{n_f}\).

2. Degré des polynômes

Raisonnement : L'équation de Bezout est \(A(p)S(p) + B(p)R(p) = P(p)\). Pour avoir un régulateur propre (ou strictement causal), on pose \(n_R = n_S\). Le polynôme \(S(p)\) s'écrit \(S(p) = p\) (pour avoir le moins de degrés possibles, \(n_S=1\)). On a donc \(n_R = 1 \implies R(p) = r_1 p + r_0\). Le polynôme caractéristique \(P(p)\) a pour degré \(\deg(P) = n_A + n_S = 1+1=2\). Ainsi, le polynôme de filtrage sera de degré 1 : \(P_f(p) = p+2\).

Reponse : Le polynôme cible est donc : \[\mathbf{P(p) = (p+4)(p+2) = p^2 + 6p + 8}\]

3. Calcul des polynômes \(R\), \(S\) et \(T\)

Raisonnement : On résout l'équation de Bezout : \[(p+1)(p) + 1 \cdot (r_1 p + r_0) = p^2 + 6p + 8\] \[p^2 + p + r_1 p + r_0 = p^2 + (1+r_1)p + r_0\] Par identification avec \(P(p)\) :

  • \(1+r_1 = 6 \implies r_1 = 5\)
  • \(r_0 = 8\)

Reponse : Les polynômes du correcteur sont donc : \[\mathbf{S(p) = p}\] \[\mathbf{R(p) = 5p + 8}\] Pour obtenir un gain statique unitaire sans modifier la dynamique de poursuite, on annule les pôles de filtrage via : \[\mathbf{T(p) = \frac{P_c(0)P_f(p)}{B(0)} = 4(p+2) = 4p+8}\] (Note : le choix \(T(p) = \frac{P(0)}{B(0)} = 8\) est également valide mais ne compense pas le pôle de filtrage dans la poursuite).

4. Transfert en boucle fermée et sensibilité

Reponse :

  • Transfert BF : \(\mathbf{\frac{Y(p)}{C(p)} = \frac{B(p)T(p)}{P(p)} = \frac{4(p+2)}{(p+4)(p+2)} = \frac{4}{p+4}}\).
  • Fonction de sensibilité : \(\mathbf{S_{yp}(p) = \frac{A(p)S(p)}{P(p)} = \frac{p(p+1)}{p^2+6p+8}}\).
Exercice 2

Exercice 2 - Structure du correcteur pour un 2nd ordre

1. Modélisation

Reponse : Le système est \(\mathbf{H(p) = \frac{K}{\frac{p^2}{\omega_n^2} + \frac{2\zeta}{\omega_n}p + 1} = \frac{B(p)}{A(p)}}\). Le degré de \(A(p)\) est \(\mathbf{n_A = 2}\).

2. Condition de rejet de perturbation

Raisonnement : On veut rejeter une perturbation de pulsation \(\omega = 1\) rad/s. La fonction de sensibilité \(S_{yp}(p)\) doit être nulle en \(p = \pm j\). Il faut donc que \(S(p)\) contienne le facteur \((p-j)(p+j) = p^2+1\). On pose \(S(p) = (p^2+1)S'(p)\).

Reponse : Le degré minimal de \(S(p)\) est donc augmenté de 2.

3. Détermination des degrés

Raisonnement : On cherche un correcteur causal (\(n_R = n_S\)). L'équation de Bezout est de degré \(\deg(P) = n_A + n_S = 2 + n_S\). Le nombre d'inconnues est \(n_{S'}\) (pour \(S'\) unitaire) \(+ n_R + 1\). Or \(n_S = n_{S'} + 2\), donc \(n_R = n_{S'} + 2\). Le nombre total d'inconnues est \(n_{S'} + n_{S'} + 2 + 1 = 2n_{S'} + 3\). En égalisant le nombre d'équations et d'inconnues : \[\deg(P) = 2 + n_S = 2 + (n_{S'} + 2) = n_{S'} + 4\] \[n_{S'} + 4 = 2n_{S'} + 3 \implies n_{S'} = 1\]

Reponse : Conclusion sur la structure :

  • Le polynôme \(S(p)\) est de degré \(1 + 2 = \mathbf{3}\).
  • Le polynôme \(R(p)\) est de degré \(\mathbf{3}\).
  • Le polynôme caractéristique \(P(p)\) sera de degré \(\mathbf{5}\).
Exercice 3

Exercice 3 - Moteur asservi en position (Régulateur RST)

1. Modélisation

Reponse : \(\mathbf{H(p) = \frac{K}{p(1+Tp)} = \frac{B(p)}{A(p)}}\) avec \(\mathbf{n_A = 2}\) et \(\mathbf{n_B = 0}\).

2. Synthèse sans erreur statique et rejet perturbation sur mesure

Raisonnement :

  • Suivi consigne constante : Implique un gain statique unitaire, obtenu par un choix approprié de \(T(p)\) (\(T(0) = P(0)/B(0)\)).
  • Rejet perturbation constante additive sur la position : C'est une perturbation de sortie. Il faut \(S_{yp}(0) = 0 \implies A(0)S(0) = 0\). Puisque \(A(p)\) contient déjà un intégrateur (\(A(0)=0\)), la condition est naturellement remplie. Aucune partie fixe n'est requise pour \(S(p)\).
  • Structure : On pose \(n_R = n_S\). \(\deg(P) = 2 + n_S\). Inconnues : \(n_S + n_R + 1 = 2n_S + 1\).

\[2 + n_S = 2n_S + 1 \implies n_S = 1\]

Reponse : Les polynômes \(R(p)\) et \(S(p)\) sont de degré 1, \(P(p)\) est de degré 3.

3. Rejet asymptotique des couples résistants

Raisonnement :

  • La perturbation \(\Gamma_r\) agit sur l'entrée du système. L'annulation de l'erreur nécessite \(B(0)S(0) = 0\).
  • Puisque \(B(0) = K \neq 0\), on doit imposer \(S(0)=0\). Le polynôme \(S\) doit contenir un intégrateur : \(S(p) = p S'(p)\).
  • Conséquence sur les degrés : \(n_S = n_{S'} + 1\). Inconnues : \(n_{S'} + n_R + 1\).

Pour \(n_R = n_S = n_{S'} + 1\), on a \(2n_{S'} + 2\) inconnues. Équations : \(\deg(P) = 2 + n_S = 3 + n_{S'}\). \[3 + n_{S'} = 2n_{S'} + 2 \implies n_{S'} = 1\]

Reponse : Les polynômes \(R(p)\) et \(S(p)\) sont de degré 2, \(P(p)\) de degré 4.

Exercice 4

Exercice 4 - Synthèse RST stricte

1. Modélisation et contraintes

Raisonnement : \(A(p) = p(1+Tp) \implies n_A = 2\). \(B(p) = G \implies n_B = 0\).

  • Strictement causal : \(n_R \le n_S - 1\).
  • Rejet \(d_1\) (sortie) : Naturellement rejeté car \(A(0)=0\).
  • Rejet \(d_2\) (entrée) : Impose \(S(0)=0 \implies S(p) = p S'(p)\).
  • Poursuite : \(P_c(p) = p^2 + 1.4p + 1\) (\(\omega_n=1\), \(\zeta=0.7\)).
  • Filtrage : Pôles à -1.

2. Détermination des degrés

Raisonnement : On pose \(n_R = n_S - 1\). Le nombre d'inconnues est \(n_{S'} + n_R + 1 = n_{S'} + (n_S - 1) + 1 = 2n_{S'} + 1\). Le nombre d'équations est \(\deg(P) = n_A + n_S = 2 + n_{S'} + 1 = n_{S'} + 3\). \[2n_{S'} + 1 = n_{S'} + 3 \implies n_{S'} = 2\] Si \(n_{S'} = 2\), alors \(n_S = 3\) et \(n_R = 2\). La condition de stricte causalité est bien respectée.

Reponse :

  • \(\deg(S) = \mathbf{3}\)
  • \(\deg(R) = \mathbf{2}\)
  • \(\deg(P) = \mathbf{5}\)

Ainsi \(\mathbf{P_f(p) = (p+1)^3}\). Les polynômes se déduisent par identification de Bezout : \(\mathbf{A(p)S(p) + B(p)R(p) = P_c(p)P_f(p)}\).

Exercice 5

Exercice 5 - Régulation de température d'une enceinte thermique

1.a - Liste des variables et des paramètres

Reponse :

  • Variables (qui évoluent avec le temps) :
  • \(Tint(t)\) : Température dans l'enceinte thermique.
  • \(Text(t)\) : Température extérieure.
  • \(Tpar(t)\) : Température dans la paroi.
  • \(Puis(t)\) : Puissance de chauffe.
  • Paramètres (constants) :
  • \(\alpha_1\), \(\alpha_2\), et \(\alpha_3\) (constantes positives).

1.b - Identification des rôles des variables

Reponse : Dans un système asservi :

  • Variable à piloter (ou grandeur réglée) : C'est ce que l'on souhaite contrôler. Ici, c'est la température interne \(Tint(t)\).
  • Variable de commande : C'est le moyen d'action pour modifier la grandeur réglée. Ici, c'est la puissance de chauffe \(Puis(t)\).
  • Perturbation : C'est un phénomène extérieur non contrôlé qui influence le système. Ici, c'est la température extérieure \(Text(t)\).

2.a - Expressions de \(H(p)\) et \(Hd(p)\)

Raisonnement : Pour exprimer les fonctions de transfert, nous devons passer dans le domaine de Laplace en supposant les conditions initiales nulles. Les équations différentielles deviennent :

  1. \(p \cdot Tpar(p) = \alpha_1(Tint(p) - Tpar(p)) + \alpha_2(Text(p) - Tpar(p)) + Puis(p)\)
  2. \(p \cdot Tint(p) = \alpha_3(Tpar(p) - Tint(p))\)

Isolons \(Tpar(p)\) dans la deuxième équation : \[Tpar(p) = \frac{p + \alpha_3}{\alpha_3} Tint(p)\]

Substituons \(Tpar(p)\) dans la première équation : \[p \frac{p + \alpha_3}{\alpha_3} Tint(p) = \alpha_1 Tint(p) - (\alpha_1 + \alpha_2) \frac{p + \alpha_3}{\alpha_3} Tint(p) + \alpha_2 Text(p) + Puis(p)\]

Multiplions tout par \(\alpha_3\) pour simplifier et regroupons les termes en \(Tint(p)\) : \[[p^2 + (\alpha_1 + \alpha_2 + \alpha_3)p + \alpha_2 \alpha_3] Tint(p) = \alpha_2 \alpha_3 Text(p) + \alpha_3 Puis(p)\]

Par principe de superposition :

  • Pour \(H(p)\), on considère que la perturbation est nulle (\(Text(p) = 0\)).
  • Pour \(Hd(p)\), on considère que la commande est nulle (\(Puis(p) = 0\)).

Reponse : \[\mathbf{H(p) = \frac{Tint(p)}{Puis(p)} = \frac{\alpha_3}{p^2 + (\alpha_1 + \alpha_2 + \alpha_3)p + \alpha_2 \alpha_3}}\] \[\mathbf{Hd(p) = \frac{Tint(p)}{Text(p)} = \frac{\alpha_2 \alpha_3}{p^2 + (\alpha_1 + \alpha_2 + \alpha_3)p + \alpha_2 \alpha_3}}\]

2.b - Gain statique de \(Hd(p)\)

Raisonnement : Le gain statique s'obtient en posant \(p = 0\) (régime permanent) : \[Hd(0) = \frac{\alpha_2 \alpha_3}{\alpha_2 \alpha_3} = 1\]

Reponse : \(\mathbf{Hd(0) = 1}\). Sens physique : Si on ne chauffe pas, à terme, la température intérieure sera exactement égale à la température extérieure.

3.a - Conclusion sur \(Hd(p)\) si enceinte parfaitement isolée

Raisonnement : "Parfaitement isolée" signifie qu'il n'y a pas d'échanges avec l'extérieur, donc \(\alpha_2 = 0\).

Reponse : Si \(\alpha_2 = 0\), alors d'après l'expression calculée précédemment, \(\mathbf{Hd(p) = 0}\). La température extérieure n'a plus d'influence sur le système.

3.b - Forme de la fonction de transfert \(H(p)\)

Raisonnement : En remplaçant \(\alpha_2 = 0\) dans \(H(p)\) : \[H(p) = \frac{\alpha_3}{p^2 + (\alpha_1 + \alpha_3)p} = \frac{\alpha_3}{p(p + \alpha_1 + \alpha_3)}\] Pour l'identifier à la forme canonique \(\frac{G}{p(1+Tp)}\), on factorise par \((\alpha_1 + \alpha_3)\) au dénominateur : \[H(p) = \frac{\frac{\alpha_3}{\alpha_1 + \alpha_3}}{p\left(1 + \frac{1}{\alpha_1 + \alpha_3}p\right)}\]

Reponse : Par identification, on trouve : \[\mathbf{G = \frac{\alpha_3}{\alpha_1 + \alpha_3} \quad \text{et} \quad T = \frac{1}{\alpha_1 + \alpha_3}}\]

3.c - Réponse indicielle de \(H(p)\)

Raisonnement : Une réponse indicielle correspond à une entrée en échelon, \(Puis(p) = \frac{1}{p}\). La sortie est donc \(Tint(p) = \frac{G}{p^2(1+Tp)}\). C'est un système avec un pôle à l'origine (comportement intégrateur) et un pôle en \(-1/T\). Sa réponse temporelle combine une rampe et une exponentielle.

Reponse : L'allure de la courbe sera une tangente horizontale à l'origine qui va tendre asymptotiquement vers une droite de pente \(G\).

3.d - Fonction de transfert en boucle fermée (FTBF)

Raisonnement : Avec un régulateur proportionnel de gain \(K\), la FTBF s'écrit : \[FTBF(p) = \frac{K \cdot H(p)}{1 + K \cdot H(p)} = \frac{\frac{K \cdot G}{p(1+Tp)}}{1 + \frac{K \cdot G}{p(1+Tp)}} = \frac{K \cdot G}{Tp^2 + p + K \cdot G}\]

Reponse : En divisant par \(K \cdot G\) pour avoir un gain unitaire au numérateur : \[\mathbf{FTBF(p) = \frac{1}{\frac{T}{K \cdot G}p^2 + \frac{1}{K \cdot G}p + 1}}\]

3.e - Erreur statique en suivi de consigne échelon

Raisonnement : Le système en boucle ouverte (\(K \cdot H(p)\)) possède un intégrateur pur (le \(p\) au dénominateur). Or, la théorie stipule qu'un système de classe 1 présente une erreur statique nulle pour une entrée en échelon.

Reponse : L'erreur statique est nulle.

3.f - Réglage pour une réponse sans dépassement la plus rapide

Raisonnement : On identifie la FTBF au modèle du second ordre : \(\frac{1}{\frac{p^2}{\omega_n^2} + \frac{2\zeta}{\omega_n}p + 1}\). On obtient \(\omega_n = \sqrt{\frac{K \cdot G}{T}}\) et \(\frac{2\zeta}{\omega_n} = \frac{1}{K \cdot G}\). Ce qui donne \(\zeta = \frac{1}{2\sqrt{K \cdot G \cdot T}}\). La réponse est la plus rapide sans dépassement (amortissement critique) pour \(\zeta = 1\). \[\frac{1}{2\sqrt{K \cdot G \cdot T}} = 1 \implies 4 \cdot K \cdot G \cdot T = 1 \implies K = \frac{1}{4 \cdot G \cdot T}\]

Reponse : En substituant \(G\) et \(T\) : \[\mathbf{K = \frac{(\alpha_1 + \alpha_3)^2}{4\alpha_3}}\]

3.g - Marge de phase de 45 degrés

Raisonnement : La fonction de transfert en boucle ouverte est \(FTBO(j\omega) = \frac{K \cdot G}{j\omega(1 + jT\omega)}\). La marge de phase est \(\Delta\phi = 180^\circ + \arg(FTBO(j\omega_c)) = 45^\circ\). L'argument est \(\arg(FTBO(j\omega)) = -90^\circ - \arctan(T\omega)\). \[-90^\circ - \arctan(T\omega_c) = -135^\circ \implies \arctan(T\omega_c) = 45^\circ \implies T\omega_c = 1 \implies \omega_c = \frac{1}{T}\] À la pulsation de coupure \(\omega_c\), le gain vaut 1 : \[|FTBO(j\omega_c)| = \frac{K \cdot G}{\omega_c \sqrt{1 + (T\omega_c)^2}} = 1\]

Reponse : En remplaçant \(\omega_c\) par \(1/T\) : \[\mathbf{K = \frac{\sqrt{2}}{G \cdot T} = \sqrt{2} \frac{(\alpha_1 + \alpha_3)^2}{\alpha_3}}\]

4.a - Approximations du 1er ordre (Enceinte avec pertes)

Raisonnement : Valeurs : \(\alpha_1=0.9\), \(\alpha_2=0.01\), \(\alpha_3=0.1\). Sur les courbes indicielles :

  • \(H(p)\) tend vers 100 en régime permanent. À \(63\%\) de la valeur finale, on lit un temps \(\tau \approx 1000\)s.
  • \(Hd(p)\) tend vers 1 en régime permanent avec la même dynamique.

Reponse : \[\mathbf{H(p) \approx \frac{100}{1 + 1000p}} \quad \text{et} \quad \mathbf{Hd(p) \approx \frac{1}{1 + 1000p}}\]

4.c - Synthèse du régulateur RST

Etape 1 : Pôles désirés \(P(p)\)
On veut un comportement du 1er ordre avec \(tr_{95\%} = 900\)s. Or \(tr_{95\%} \approx 3\tau_{BF}\), donc \(\tau_{BF} = 300\)s. Le pôle de poursuite est \(P_c(p) = 1 + 300p\). Tous les pôles doivent être égaux, donc les pôles de filtrage seront aussi sous la forme \(1 + 300p\).

Etape 2 : Structure de R(p) et S(p)
Rejet asymptotique d'un échelon de perturbation sur \(Text\). Le régulateur doit contenir le modèle de la perturbation (un intégrateur). Donc \(R(p)\) doit contenir le facteur \(p\). Pour assurer la stricte causalité \(\deg(R) > \deg(S)\) et pouvoir résoudre l'équation de Bézout, on choisit : \[R(p) = p(r_1 p + r_0) \implies \deg(R)=2\] \[S(p) = s_1 p + s_0 \implies \deg(S)=1\] L'équation \(A(p)R(p) + B(p)S(p)\) sera de degré 3. Le polynôme désiré \(P(p)\) doit donc être de degré 3 : \[P(p) = (1 + 300p)^3 = 27\,000\,000 p^3 + 270\,000 p^2 + 900 p + 1\]

Etape 3 : Résolution de l'équation de Diophante (Bézout)
\[(1000p + 1)(r_1 p^2 + r_0 p) + 100(s_1 p + s_0) = 27\,000\,000 p^3 + 270\,000 p^2 + 900 p + 1\] En identifiant les coefficients :

  • \(p^3\) : \(1000 r_1 = 27\,000\,000 \implies r_1 = 27\,000\)
  • \(p^2\) : \(1000 r_0 + r_1 = 270\,000 \implies r_0 = 243\)
  • \(p^1\) : \(r_0 + 100 s_1 = 900 \implies s_1 = 6.57\)
  • \(p^0\) : \(100 s_0 = 1 \implies s_0 = 0.01\)

Reponse : Les polynômes du régulateur sont : \[\mathbf{R(p) = p(27000p + 243)} \quad \text{et} \quad \mathbf{S(p) = 6.57p + 0.01}\]

Etape 4 : Détermination de T(p)
On veut \(FTBF = \frac{B(p)T(p)}{P(p)} \equiv \frac{1}{1+300p}\). \[\frac{100 \cdot T(p)}{(1+300p)^3} = \frac{1}{1+300p} \implies 100 \cdot T(p) = (1+300p)^2 = 90000p^2 + 600p + 1\]

Reponse (suite) : \[\mathbf{T(p) = 900p^2 + 6p + 0.01}\]

4.d - Fonction de sensibilité

Raisonnement : La fonction de sensibilité par rapport à une perturbation en sortie se définit par : \[S(p) = \frac{A(p)R(p)}{P(p)} = \frac{(1+1000p) \cdot 243p \cdot (1 + \frac{27000}{243}p)}{(1+300p)^3} \approx \frac{243p \cdot (1+1000p) \cdot (1+111.11p)}{(1+300p)^3}\]

Analyse asymptotique pour le diagramme de Bode :

  • Basses fréquences : Le terme \(243p\) domine (+20 dB/décade). Garantit le rejet asymptotique d'un échelon (gain \(\to -\infty\) dB quand \(\omega \to 0\)).
  • Cassures :
  1. Zéro à \(\omega_1 = 0.001\) rad/s \(\implies\) Pente +40 dB/déc.
  2. Pôle triple à \(\omega_2 \approx 0.0033\) rad/s \(\implies\) Pente -20 dB/déc.
  3. Zéro à \(\omega_3 \approx 0.009\) rad/s \(\implies\) Pente 0 dB/déc.
  • Hautes fréquences : Le gain tend vers \(\frac{1000 \cdot 27000}{300^3} = 1\), soit 0 dB.

Reponse : Le tracé présentera un rejet parfait en statique et terminera à 0 dB en hautes fréquences.
